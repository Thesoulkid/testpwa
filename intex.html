Demo PWA — Installable Web App

Use this page to test PWA install flow on Android and "Add to Home Screen" instructions for iOS.

On Android Chrome: you should either see a browser install prompt (three-dot menu → Install app) or you can tap the blue button above.
On iPhone: follow the instructions shown below.
To install on iPhone / iPad:
  1. Tap the Share icon (the square with an arrow).
  2. Choose Add to Home Screen.
  3. Tap Add — the site will behave like an app.
